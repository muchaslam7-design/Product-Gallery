## Key Features
* **Dynamic Grid Layout:** A fully responsive gallery that adjusts seamlessly from mobile screens to 4K displays using CSS Grid and Flexbox.
* **Interactive Hover States:** Smooth CSS transitions and scale effects to enhance user engagement.
* **High-Resolution Image Optimization:** Built to handle high-quality product photography with optimized loading times.
* **Smart Filtering:** Category-based navigation to sort products instantly without page reloads.
* **Modal View Integration:** Click-to-expand functionality for detailed product inspection (LightBox effect).
* **Clean & Modular Code:** Highly reusable components structured for easy integration into any React project.
* **Real-Time Status Indicator:** Features a dynamic presence system that detects user connectivity. It automatically displays an **"Online"** status when the user is active and switches to **"Offline"** when disconnected,                                    ensuring a transparent user experience.
* **Live Connectivity Monitoring:** Built using React hooks to listen for browser connection events, providing instant feedback without page refreshes.

## Tech Stack
* **Core:** React.js (Hooks & Functional Components)
* **Styling:** Tailwind CSS / CSS Modules
* **Icons:** Lucide-React / FontAwesome
